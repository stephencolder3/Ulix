# Ulix

Ulix is an open-source display typeface designed for modern interfaces, branding, and editorial use.

## Styles
- Ulix Heavy

## Usage
You are free to use Ulix for:
- Personal and commercial projects
- Websites, apps, and branding
- Print and digital media

You may also modify and redistribute the font under the terms of the license.
